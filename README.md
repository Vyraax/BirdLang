## Setup

```bash
git clone --recursive https://github.com/Vyraax/BirdLang.git
```

###### Linux

```bash
./setup.sh
```

###### Windows

```bash
./setup.bat
```

## Compilation

###### Linux

```bash
make
```

###### Windows
```bash
Open the Solution (.sln) file and click on the compile button.
```
