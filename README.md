## Compile from sources

###### Linux build

```console
git clone https://github.com/Vyraax/BirdLang.git
cd BirdLang
./setup.sh
make
```

###### Windows build

```console
git clone https://github.com/Vyraax/BirdLang.git
cd BirdLang
./setup.bat
```

Open the generated solution file (.sln), and click on the compile button.
